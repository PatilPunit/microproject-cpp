##**🎮 Pokémon Console Game (C++)**


**A text-based Pokémon game built using C++ that simulates battles, catching Pokémon, leveling up, and gym fights.**

**🚀 Features**

⚔️ Turn-based battle system
🎯 Catch wild Pokémon using Pokéballs
📈 Level up and evolution system
🎒 Inventory system (Pokéballs, potions, items)
🏙️ Multiple cities with Gym Leaders
🔄 Switch Pokémon during battle
🎲 Random encounters

**🧠 Concepts Used**

Object-Oriented Programming (OOP)
Classes & Objects
Inheritance (GymLeader → Brock, Misty, etc.)
Polymorphism (virtual functions)
STL (Standard Template Library)
vector
Game logic design
Randomization (rand())

**🎮 Gameplay**

Create your character
Choose your starter Pokémon
Explore areas (Forest, Cave, Lake)
Battle wild Pokémon
Catch and train your team
Challenge Gym Leaders

**🛠️ How to Run**

Compile:
g++ main.cpp -o pokemon

Run:
./pokemon
